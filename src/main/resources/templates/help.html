Назначение приложения:

Приложение предназначено для формирования списка аксессуаров для кабельной продукции. Входными данными является txt файл со списком кабельной продукции (обозначение кабеля в проекте, наименование кабеля, длина (м)).

Возможности приложения:

  • Добавление в базу данных: отдельного кабеля (Главная страница->Редактирование БД->Добавить отдельный кабель), несколько кабелей (Главная страница->"Редактирование БД"->"Добавить кабели из файла")
  • Удаление выбранного кабеля из базы данных: Главная страница->Редактирование БД->Удалить (напротив соответствующего кабеля)
  • Добавление списка кабелей для подбора аксессуаров (Главная страница->Начать работу->Добавить файл со списом кабеля)
